== README

This app basically aims at making a playlist that will satisfy a group of people using their facebook likes and music charts, it also gives user the ability to create dynamic playlist(guests can change the order of the songs on the server using their phones).

dependencies =>
Rails version 4.0.0
Itunes gem 
mini_fb gem
httparty gem
koala gem
google api client gem 
xmpp4r_facebook gem
